# 🚤 boAt Sales Analysis Dashboard with Machine Learning

An end-to-end Business Intelligence project developed using **Power BI**, **Python**, and **Machine Learning** to analyze boAt sales performance, customer behavior, product performance, regional trends, and predict loss-making orders.

---

# 📌 Project Overview

This project demonstrates the complete Business Intelligence workflow from raw data to predictive analytics.

The solution includes:

- Data Cleaning using Power Query
- Data Modeling
- Advanced DAX Calculations
- Interactive Power BI Dashboards
- Exploratory Data Analysis (Python)
- Machine Learning Model
- Prediction Dashboard

---

# 🎯 Business Objectives

- Analyze overall sales performance
- Identify top-performing products
- Understand customer purchasing behavior
- Evaluate regional sales performance
- Analyze sales trends
- Detect factors affecting profitability
- Predict loss-making orders using Machine Learning

# 🤖 Machine Learning

## Business Problem

Predict whether an order will result in a loss.

### Features Used

- Quantity
- Discount Percentage
- Gross MRP
- Shipping Cost
- Delivery Days
- Festival Period
- Weekend
- Hero Product

### Algorithm

Random Forest Regressor

### Model Performance

| Metric | Score |
|----------|-----------|
| MAE | 0.0138 |
| MSE | 0.000256 |
| RMSE | 0.0160 |
| R² Score | 0.99999999998 |

---

# 📊 Key Business Insights

- True Wireless Earbuds generated the highest sales.
- Hero products contributed the majority of revenue.
- Online Marketplace was the highest-performing sales channel.
- Tier-1 customers generated the highest revenue.
- Discount percentage significantly impacted profitability.
- Nearly 29% of orders resulted in losses.
- Shipping cost and discounts were major factors influencing loss-making orders.
- Machine Learning successfully identified potential loss-making orders with excellent accuracy.
